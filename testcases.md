# 🧪 Тест-кейсы для Smoke-прогона (Mentorpiece FTB)

---

### 🟢 Модуль: Доступность главной страницы
**ID:** ID_1  
**Приоритет:** `Critical` 🔴


| Параметр | Описание |
| :--- | :--- |
| **Предусловия** | Стабильное интернет-соединений на ПК |
| **Ожидаемый результат** | Страница открылась без ошибок |

#### Шаги:
1. Открыть любой браузер на своем ПК
2. Открыть страницу сайта по адресу URL: https://ftb.mentorpiece.org/

> **Вложения:** скрины, видео

---

### 🔵 Модуль: Регистрация пользователя
**ID:** ID_2  
**Приоритет:** `Critical` 🔴


| Параметр | Описание |
| :--- | :--- |
| **Предусловия** | 1. Главная страница открылась без ошибок |
| **Ожидаемый результат** | аккаунт создан и готов к работе в системе |

#### Шаги:
1. Открыть главную страницу сайта URL: https://ftb.mentorpiece.org/
2. Нажать кнопку «Войти» в форме на сайте либо в верхнем меню справа
3. Нажать кнопку внизу страницы «Зарегистрируйся здесь»
4. Заполнить поля со звездочкой: имя, фамилия, имя пользователя, адрес электронной почты, пароль.
5. Нажать кнопку «Создать аккаунт»

> **Вложения:** скрины, видео
